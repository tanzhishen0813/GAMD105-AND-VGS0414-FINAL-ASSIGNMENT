# 美食查找应用（Food Finder Application）
本项目是网页设计课程的期末作业，应用使用 Bootstrap 5 与 JavaScript 构建，提供美食浏览、菜谱查看与店家推荐功能。

---

## 📌 功能说明

### 1. 首页（index.html）
- 简单介绍应用并提供进入按钮。

### 2. 搜索页面（search.html）
- 显示 9 个马来西亚特色食物  
- 每个 food card 可查看：
  - 菜谱说明  
  - 推荐店家  
- 使用 Bootstrap Modal 显示内容

### 3. 关于页面（about.html）
- 显示两位组员简介

### 4. 联系页面（contact.html）
- 联系表单  
- JavaScript 表单验证  
- 联系方式展示

### 5. style.css
- 自定义样式、card 图片统一格式

### 6. script.js
- 动态生成美食卡片  
- Modal 动态内容  
- 联系表单验证  

---

## 💻 使用技术
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript (ES6)

---

## 📱 响应式设计
所有页面使用 Bootstrap 5，兼容手机与电脑。
